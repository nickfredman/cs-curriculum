# 11. Mobile Development

**Prerequisites**:

- Programming Fundamentals
- UI/UX Design Basics
- Web Development
  **Estimated Time**: 12-14 weeks (10-15 hours/week)

## Learning Objectives

- Master mobile development
- Understand mobile platforms
- Learn app architecture
- Gain practical mobile experience
